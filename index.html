Key Concepts

Stage 0: The Basics the Web & HTML

Web

Web is invented in 1990s and it contains of 30 billion pages.

Web is a series of HTML (Hyper Text Markup Language) document.

For the process of web navigation, there are 3 main roles which are user with browers, the internet, and the servers.

The search request will be delivered through the internet to the servers ("storages" that contain extensive amount of web pages), using the protocal of http. The servers will then request the internet to respond to the user's search request. Information respond base on the request will then be displayed on user's digital devices.

HTML

HTML is the most fundamental type of document for all web pages that helps to link everything together. The link that helps to connect all the media contents called the Hyper Link.

HTML markup is made up of tags and tags that has no content such as image doesn not need a closing tag and this is known as a void tag.

The whole line that is created with tags called element.

Inline vs Block

HTML elements are either inline or block. Block creates an "invisible box" around the content.

For example, br (inline) creates a space between words or to separate one line into two lines. p (block) creates an invisible box with height and width around text or sentence.

Inline tags: b, br, img, em, span, strong

Block tags: h, t, p, div, form

span & div

span and div both have content.

span is inline element; div is block element.

Stage 1: Structure HTML Document & Add Style via CSS

The "branching structure" of HTML

"tree-like structure" of HTML meaning HTML elements can contain multiple elements within them. Such relationship can be described as a family tree.

The structure will present itself as a series of nested boxes in the browser, meaning there are multiple boxes within boxes.

HTML Indentations

A series of changing HTML identations on the left from the beginning to the end can be seen in a HTML document. The more indented an element is the more "boxes" it contains.

Text Editors for Programming

Subline text for example can allow more efficiency when it comes to code writing. Because the editor will automatically provide a closing tag when an opening tag is written.

Group elements via div & class

Repetition like writting the same codes multiple times may lead to errors when changing something in a document. To avoid repetition, the method of grouping elements via div and class can be used.

CSS

CSS/ Cascasing Style Sheets serve to visualize the presentation of an HTML document by adding style and color into it.
